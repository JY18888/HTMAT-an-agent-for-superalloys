# HTMAT: A Knowledge-Data Dual-Driven AI Agent for Reliable Reasoning and Autonomous Design of Superalloys


HTMAT is a revolutionary AI agent that combines metallurgical knowledge graphs with data-driven machine learning to enable trustworthy, autonomous design of nickel-based single crystal superalloys. By integrating domain-specific constraints with advanced AI reasoning, HTMAT eliminates hallucinations while dramatically accelerating materials discovery.

## 🌟 Key Features

- **Knowledge-Grounded Reasoning**: Integrates a comprehensive superalloy knowledge graph derived from authoritative handbooks to suppress LLM hallucinations
- **Dual-Driven Architecture**: Combines structured knowledge retrieval with neural network-guided virtual screening
- **Autonomous Design Pipeline**: End-to-end capability from user requirements to experimentally validated alloys
- **Self-Evolving System**: Creates a knowledge flywheel where experimental results continuously enrich the knowledge base
- **Physically Plausible Predictions**: Ensures all recommendations adhere to fundamental metallurgical principles

## 🛡️ Data Security
HTMAT uses a compact 8-billion-parameter LLM variant (Qwen3-8B) that enables fully localized, offline deployment on high-performance workstations, ensuring absolute data privacy for proprietary alloy research.
